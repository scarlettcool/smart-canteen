# Sprint 2: 消费管理权限点清单

| 权限编码 (Backend Code) | 描述 | 控制范围 |
| :--- | :--- | :--- |
| `CONSUME_ACCOUNT_READ` | 查看账户列表与余额 | 账户管理、报表中心 |
| `CONSUME_ACCOUNT_ADJUST` | 手动调账/充值/冻结 | 账户管理操作按钮 |
| `CONSUME_TRADE_READ` | 查看交易流水明细 | 交易记录、个人流水 |
| `CONSUME_TRADE_CORRECT` | 执行交易冲正 | 交易管理操作按钮 |
| `CONSUME_REFUND_AUDIT` | 审核退款申请 | 退款审核页面 |
| `CONSUME_APPEAL_HANDLE` | 处理申诉申请 | 失约申诉页面 |
| `CONSUME_REPORT_VIEW` | 查看并导出报表 | 所有报表路由 |
| `CONSUME_CONFIG_WRITE` | 修改限额、补贴规则 | 消费设置页面 |
| `CONSUME_DEVICE_MANAGE` | 维护设备、下发指令 | 设备管理页面 |
