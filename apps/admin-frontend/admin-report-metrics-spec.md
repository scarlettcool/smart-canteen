# Sprint 2: 财务统计报表口径说明书

## 1. 核心公式定义
*   **消费总额 (Gross Consumption)**: 所有成功支付状态的单据金额累加（不扣除退款）。
*   **退款/冲正总额 (Total Refunds)**: 状态为 `Corrected` 或 `Refunded` 的单据原值累加。
*   **实收净额 (Net Revenue)**: `Gross Consumption` - `Total Refunds`。
*   **客单价 (AOV)**: `实收净额` / `成功消费笔数`。
