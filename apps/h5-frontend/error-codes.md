# 智慧食堂错误码定义 (Error Catalog)

## 1. 通用系统错误 (Global)
| 错误码 | 业务含义 | 前端反馈方式 | 推荐 CTA |
| :--- | :--- | :--- | :--- |
| `NET_ERROR` | 网络连接异常 | Toast | 重试 |
| `TIMEOUT` | 请求超时 | Toast | 刷新页面 |
| `SERVER_ERROR` | 后端服务崩溃 | PageError | 咨询客服 |
| `UNAUTHORIZED` | 会话过期/未登录 | Redirect | 重新登录 |
| `FORBIDDEN` | 权限不足 | Dialog | 申请权限 |

## 2. 身份认证与权限 (Auth: E1xxx)
| 错误码 | 业务含义 | 前端反馈方式 | 推荐 CTA |
| :--- | :--- | :--- | :--- |
| `E1001` | 员工编号不存在 | Inline | 重新输入 |
| `E1002` | 注册申请审核中 | Page (Status) | 查看进度 |
| `E1003` | 注册申请被驳回 | Page (Status) | 修改资料 |

## 3. 钱包与交易 (Wallet: E2xxx)
| 错误码 | 业务含义 | 前端反馈方式 | 推荐 CTA |
| :--- | :--- | :--- | :--- |
| `E2001` | 余额不足 (SOT Fail) | Dialog | 立即充值 |
| `E2002` | 充值渠道关闭 | Toast | 换个金额 |
| `E2003` | 幂等请求 ID 重复 | Toast | 忽略 |
| `E2004` | 支付密码错误 | Dialog | 找回密码 |

## 4. 业务逻辑与预约 (Biz: E3xxx)
| 错误码 | 业务含义 | 前端反馈方式 | 推荐 CTA |
| :--- | :--- | :--- | :--- |
| `E3001` | 预约名额已满 | Toast | 换个日期 |
| `E3002` | 不在预约时间内 | Dialog | 我知道了 |
| `E3003` | 存在未处理的违约 | Dialog | 去申诉 |
| `E3004` | 档口已闭餐 | Toast | 选择其他食堂 |

## 5. 交互契约说明
- **Toast**: 3秒自动消失，用于轻微提示。
- **Dialog**: 强阻塞，必须包含至少一个 Action 按钮。
- **PageError**: 整个页面状态占位（StateLayout），用于数据初始化失败。
- **Inline**: 字段下方的红色文字，用于表单校验。
